Rojo-ready Roblox prototype source.

Latest notes:

- `docs/NON_WORLD_UPDATE.md` covers melee/equipment/weapon/progression updates.
- `docs/PARKOUR_TUNING_UPDATE.md` covers movement, wall climbing, slide, aim glide, camera, animation, and tuner updates.
