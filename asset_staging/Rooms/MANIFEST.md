# Rooms

Expected Studio asset shape:

- Room templates live in `ServerStorage.ServerAssets.Rooms.<TemplateId>`.
- Decorations live in `ServerStorage.ServerAssets.Decorations.<DecorationId>`.
- Room templates should include a `RoomSpawn` part when precise placement matters.
- Decorations should have sensible pivot/origin placement for room editing.

This folder is for notes and import planning only.
